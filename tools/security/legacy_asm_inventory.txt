#   path | subsystem | risk | status | replacement
#     path        - repo-relative, forward-slash
#     subsystem   - owning area
#     risk        - high | medium | low  (security criticality of migrating it)
#     status      - legacy (migration pending) | migrating | <unused>
#     replacement - target NHL/NexusHLK module, or TBD

#     failure: new first-class work must be NHL/NexusHLK.
#   - A listed file that no longer exists is a stale entry: prune the line. The
#     inventory only shrinks as migration deletes legacy modules.
#   - A non-comment line without the pipe-delimited columns is malformed.
